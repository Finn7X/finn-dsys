---
title: "Git Worktree: 同时在多个分支上工作"
date: 2026-03-08
tags:
  - git
---

`git worktree` 允许你在同一个仓库中同时检出多个分支到不同的目录，不需要多次 clone。

```bash
# 创建一个新的工作目录
git worktree add ../feature-branch feature-branch

# 列出所有工作目录
git worktree list

# 删除工作目录
git worktree remove ../feature-branch
```

适合需要同时对比/开发两个分支的场景，比如修复紧急 bug 时不需要 stash 当前工作。
