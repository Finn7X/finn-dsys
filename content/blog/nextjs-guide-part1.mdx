# 项目搭建与基础概念

Next.js 是一个基于 React 的全栈 Web 框架，它提供了服务端渲染、静态生成、API 路由等开箱即用的功能。

## 创建项目

使用 `create-next-app` 快速初始化项目：

```bash
npx create-next-app@latest my-app --typescript --tailwind --app
```

## App Router 核心概念

App Router 是 Next.js 13+ 引入的新路由系统，基于文件系统约定：

- `page.tsx` - 路由页面
- `layout.tsx` - 共享布局
- `loading.tsx` - 加载状态
- `error.tsx` - 错误边界

## 文件系统路由

```
app/
├── page.tsx          → /
├── about/
│   └── page.tsx      → /about
└── blog/
    ├── page.tsx      → /blog
    └── [slug]/
        └── page.tsx  → /blog/:slug
```

每个目录中的 `page.tsx` 文件都会自动成为一个可访问的路由。
