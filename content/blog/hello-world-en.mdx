## Welcome

Hello, welcome to **Finn Days**! This is my personal tech blog where I share web development practices, technical insights, and open source projects.

## Tech Stack

This blog is built with the following technologies:

- **Next.js 16** — React full-stack framework
- **TypeScript** — Type safety
- **Tailwind CSS 4** — Utility-first CSS
- **Velite** — MDX content management
- **shadcn/ui** — UI component library
